Evolu.js

Evolu is a programming language to automatically generate programs by evolution (genetic programming). It is created to be readable by human beings (instead of artificial neural networks) and easily editable and mixable for genetic algorithm (instead of tree structure and modern production languages).

Evolu.js is a JavaScript implementation of virtual machine to execute programs on Evolu.
