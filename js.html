Evolu.js

Evolu is a programming language to automatically generate programs by evolution (genetic programming). It create to be readable for human (instead of artificial neural networks) and easy editable and mixable for genetics algorithm (instead of tree structure in modern production languages).

Evolu.js is a JavaScript implementation of virtual machine to execute programs on Evolu.
